# Research and Synthesis Reference

## Source Evaluation

| Criteria | ✅ Good | ❌ Bad |
|----------|---------|--------|
| Recency | < 1 year | > 2 years |
| Authority | Official docs, core team | Random blogs |
| Relevance | Same stack version | Different framework |
| Depth | Explains why | Just shows code |
